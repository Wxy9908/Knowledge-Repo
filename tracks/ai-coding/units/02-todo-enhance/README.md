# 练习 2 · 添加 / 删除 Todo

> 在练习 1 基础上，为 TodoList 增加交互能力。  
> 建议 Prompt 数：**1 个**（预计 20～30 分钟）

---

## 需求文档 v1.0

**1. 概述**  
在现有 `/todos` 静态列表上，支持 **添加** 和 **删除** todo。

**2. 添加**  
- 输入框 +「添加」按钮  
- 输入标题，点击添加后，列表多一条  
- 新 todo 默认 `done: false`，显示「未完成」  
- **标题为空时不添加**（可不做 Toast，静默忽略即可）

**3. 删除**  
- 每条 todo 右侧有「删除」按钮  
- 点击后从列表移除

**4. 技术要求**  
- 状态用组件内 `ref`（**不用 Pinia**）  
- 继续沿用 `{ id, title, done }` 结构  
- 只改 `src/components/TodoList.vue`（除非你认为必须改其他文件，需说明理由）  
- 不引入新依赖

**5. 不在范围**  
- 编辑 / 切换完成状态  
- Pinia、localStorage、接口  
- 确认弹窗、精致样式
