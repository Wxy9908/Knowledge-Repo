# 身心修炼 · 日常关怀

> 个人专属模块 · 与技术分析轨道分开 · 由 AI 以心理成长陪伴者身份协助

## 怎么用

1. **先填** [`notes/intake.md`](./notes/intake.md) — 让我了解你的基线（可只写愿意分享的部分）
2. **读** [`notes/framework.md`](./notes/framework.md) — 四象限觉察框架
3. **每天** 用 [`notes/daily-checkin.md`](./notes/daily-checkin.md) 打卡，或打开 [28 天计划](/tracks/mind-body/schedule)
4. **对话** 在 Cursor 里直接聊——身体、情绪、睡眠、压力、关系、习惯都行；要点记入 [`notes/session-log.md`](./notes/session-log.md)
5. **每周** 花 10 分钟回看打卡，只问一个问题：「哪一件小事让我感觉稍微好了一点？」

## 文件说明

| 文件 | 用途 |
|------|------|
| `brief.md` | 轨道简介 |
| `notes/framework.md` | 修炼理论框架 |
| `notes/intake.md` | 初次了解 / 基线 |
| `notes/daily-checkin.md` | 每日打卡模板 |
| `notes/session-log.md` | 陪伴对话摘要 |
| `schedule.yaml` / `schedule.md` | 28 天计划 |

## 与我对话时可以问

- 「今天我状态很差，帮我理一理」
- 「我想改善睡眠 / 焦虑 / 拖延，从哪里开始」
- 「帮我写今天的打卡」
- 「这周复盘一下我的打卡记录」

我会以**资深心理成长陪伴者**的身份回应：温和、具体、可执行，不过度诊断，不制造焦虑。
