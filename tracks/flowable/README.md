# Flowable 流程对接（前端）

流程**搭建**由后端负责；本轨道帮你掌握**对接审批流**所需的前端能力。

## 快速入口

| 文档 | 说明 |
|------|------|
| [学习路线](./roadmap.md) | 阶段目标、能力地图、职责边界 |
| [**21 天学习计划（平台）**](http://localhost:5173/tracks/flowable/schedule) | 每日任务 · 资源链接 · 打卡自测 · **薄弱点自动巩固** |
| [21 天学习计划（Markdown）](./schedule.md) | IDE 内阅读版（与平台同步） |
| [轨道简介](./brief.md) | 一句话定位 |

> 平台链接需先 `pnpm dev` 启动 Dashboard。也可从首页 → Flowable 卡片 → 轨道详情 → **打开学习计划**。

## 定制化学习怎么用

1. 在平台学习计划页完成每日任务勾选
2. 自测项诚实标记 **掌握 / 不熟 / 跳过**
3. 标记「不熟」的内容会在**后续聚焦日**顶部出现「针对性巩固」
4. 打卡后可看「明日预告」——提前知道明天要补什么
5. 进度存在浏览器本地，可跳天、多学、少学，不必严格按日历
